# UI 前端设计文档

## 1. 设计风格

### 1.1 设计理念

采用现代 **SaaS 管理后台** 风格，以深色主题为主、支持明暗切换。设计灵感来源于 Vercel Dashboard、Linear、Grafana 等产品的视觉语言。

### 1.2 设计原则

| 原则 | 说明 |
|---|---|
| **信息密度适中** | 在不拥挤的前提下呈现更多有效信息 |
| **实时感** | 进度条、速度数字实时跳动，增强操控感 |
| **操作即时反馈** | 按钮点击即变色，操作即有 toast 提示 |
| **响应式** | 适配桌面宽屏与平板设备 |

### 4.1 侧边栏 (Sidebar)

- 固定宽度 **220px**，支持折叠到 **64px** (仅图标)

- 导航项带图标 + 文字，选中态左侧有 3px 主题色指示条
- 底部：折叠按钮

### 4.2 顶栏 (Header)

- 高度 **56px**，毛玻璃背景 (`backdrop-filter: blur(12px)`)
- 左侧：面包屑 / 页面标题
- 右侧：系统状态 (CPU/内存/下载速度) + 主题切换按钮

## 6. 组件规范

### 6.1 按钮

| 类型 | 样式 | 典型用途 |
|---|---|---|
| Primary | 主题色填充 + 白色文字 | 新建任务、保存 |
| Secondary | 透明底 + 主题色边框文字 | 取消、重置 |
| Danger | 红色填充 | 删除 |
| Ghost | 透明底 + 灰色文字，hover 显色 | 操作列图标按钮 |
| Icon | 圆形/方形纯图标 | 播放、暂停、关闭 |

### 6.2 状态标签 (Badge)

| 状态 | 颜色 | 文字 |
|---|---|---|
| 排队中 | `--warning` | QUEUED |
| 下载中 | `--info` | DOWNLOADING |
| 合并中 | `--accent` | MERGING |
| 已完成 | `--success` | COMPLETED |
| 错误 | `--danger` | ERROR |
| 已停止 | `--text-secondary` | STOPPED |

### 6.3 动效规范

| 场景 | 动画 | 参数 |
|---|---|---|
| 页面切换 | Fade + SlideUp | 200ms ease-out |
| 卡片悬停 | scale + shadow | 150ms ease |
| 进度条 | 流光效果 | linear infinite |
| 数字变化 | 数字翻转 | 300ms spring |
| Toast 通知 | SlideIn from right | 200ms ease-out |
